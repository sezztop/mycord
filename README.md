# MyCord — Discord-style clone 🟣

Свой клон Discord на **React + Vite**, захостенный на **GitHub Pages**.

> Это фронтенд-демо (Этап 1): красивый интерфейс в стиле Discord с локальным чатом.
> Сообщения сохраняются в браузере (localStorage). Реальный чат между друзьями
> (через Firebase) — следующий этап.

## ✨ Что уже есть
- Боковая панель серверов и каналов (текстовые + голосовые)
- Окно чата: сообщения, аватары, группировка, метки времени, разделители дней
- Отправка сообщений (Enter) + случайные ответы бота
- Список участников с ролями и статусами (в сети / не активен / не беспокоить / не в сети)
- Тёмная тема Discord, адаптив под телефон
- Поддержка блоков кода ```...```

## 🚀 Запуск локально
```bash
npm install
npm run dev
```

## 🏗️ Сборка
```bash
npm run build
```

## 📦 Деплой
Пуш в ветку `main` автоматически собирает и публикует сайт на GitHub Pages
через GitHub Actions (`.github/workflows/deploy.yml`).
